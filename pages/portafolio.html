PROYECTO AQUAVET

Aquavet es un sistema diseñada para la gestión de una clínica
veterinaria, en su desarrollo se utilizo Angular para la
interfaz de usuario, .NET para el backend y SQL Server para
la base de datos. Este sistema proporciona una plataforma
intuitiva y eficiente que facilita la administración de
pacientes, el seguimiento de tratamientos, la gestión de
citas y el manejo de inventarios

ANALIZADOR LEXICO

Este proyecto es un analizador léxico desarrollado en el
entorno de Apache NetBeans utilizando Java. Su objetivo es
analizar cadenas de texto para descomponerlas en tokens
significativos, que luego pueden ser utilizados en procesos
de compilación o interpretación. El analizador léxico
identifica elementos como palabras clave, identificadores,
operadores y otros componentes sintácticos dentro del
código fuente.

TEOREMAS

Este proyecto, está desarrollado en el entorno de Apache
NetBeans utilizando Java, se centra en la resolución de
ecuaciones matemáticas. Los usuarios pueden ingresar sus
propias ecuaciones, y el programa procesa y resuelve estas
ecuaciones, proporcionando soluciones precisas. Está diseñado
para manejar una variedad de ecuaciones, desde las más simples
hasta las más complejas.

CARACTERISTICAS DE WINDOWS

Este programa, desarrollado en Visual Studio con C#, está
diseñado para ofrecer una visión detallada de las
características de un sistema Windows. Es capaz de
inspeccionar y mostrar información clave sobre el hardware,
el software, los procesos en ejecución, el rendimiento del
sistema, las conexiones de red, y otros aspectos críticos.
Con una interfaz intuitiva, el programa permite a los usuarios
monitorizar y gestionar de manera efectiva los recursos y el
estado del sistema operativo Windows.

CLINICA DENTAL

Este programa, desarrollado en Visual Studio con C++, es un
sistema diseñado para una clínica dental. Implementa una
estructura de nodos utilizando árboles, optimizando así la
velocidad y la compatibilidad del sistema. Este enfoque
permite una gestión eficiente de los datos y una rápida
respuesta en las operaciones, asegurando un rendimiento
superior en entornos clínicos donde la precisión y la rapidez
son esenciales.
